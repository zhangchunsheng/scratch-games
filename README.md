# Scratch 转 HTML 工具

将 Scratch 3.0 项目（.sb3 文件）转换为可在浏览器中独立运行的 HTML 游戏。

## 使用方式

1. 打开 `scratch_to_html_converter.html` 或 `htmlifier-offline.html`
2. 拖拽或点击上传你的 `.sb3` 文件
3. 点击「下载 HTML」获取转换后的游戏文件
4. 在浏览器中打开生成的 HTML 即可运行

## 支持的功能

- **运动类积木**：移动、旋转、坐标定位、边界弹射
- **外观类积木**：说话/思考气泡、切换造型、大小、特效
- **事件类积木**：绿旗、按键、广播
- **控制类积木**：循环、条件、等待、重复直到
- **运算类积木**：数学运算、比较、随机数
- **变量**：读取/设置

## 已知局限

- 声音积木暂未实现
- 克隆功能为基础版本
- 视频侦测和自定义积木暂不支持

## 文件说明

- `scratch_to_html_converter.html` - 现代化 UI 的转换器
- `htmlifier-offline.html` - HTMLifier 离线版本
- `CLAUDE.md` - 项目开发指南
